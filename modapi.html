Docs

Welcome to Isotopeia's Modding API docs! Please note that every example and the mods are purely written in JSON.

{}

This object contains the entirety of the mod(s).

"buildings":{"e":[]}

Houses the list to add Electron buildings.

"buildings":{"en":[]}

Houses the list to add Electron Neutrino buildings.

"buildings":{"(e(n)/u)":[{}]}

Houses an (Electron (Neutrino)/ Up Quark) building. Properties:

price: int

The price of the building. erties:

name: String

The name (any characters, recommended format = Sentence case) of the building. erties:

perSecond: int

The production/second of the building. erties:

id: String

The identifier of the building. Should be in the camelCase format.

"customBehaviors":""

Allows for custom JavaScript to be used. Instead of using your preferred quoting, you have to use another quoting method in the JavaScript, sadly.

Example mod

We'll create an example mod. We start with an object: {} Then we add a buildings property, and the Electron/Electron Neurino lists: {     "buildings": {         "e": [                  ],         "en": [                  ]     } } Then, the example "Electron-quark constructor". This part is pretty self explanatory. Define the details: {     "buildings": {         "e": [             {                 "price": 1000,                 "name": "Electron-quark constructor",                 "perSecond": 15,                 "id": "electronQuarkConstructor"             }         ],         "en": [                  ]     } } We can then finally make a "Electron neutrino-quark constructor": {     "buildings": {         "e": [             {                 "price": 1000,                 "name": "Electron-quark constructor",                 "perSecond": 15,                 "id": "electronQuarkConstructor"             }         ],         "en": [             {                 "price": 1000,                 "name": "Electron neutrino-quark constructor",                 "perSecond": 15,                 "id": "electronNeutrinoQuarkConstructor"             }         ]     } } Done! We now have a functional mod for Isotopeia! This mod adds quark constructors.
